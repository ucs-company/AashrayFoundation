# Plan International Canada - Website Clone

Yeh Plan International Canada (plancanada.ca) ki ek HTML/CSS/JS replica hai.

## Files
- `index.html` — Main homepage
- `style.css`  — Saari styles (responsive)
- `main.js`    — Scroll animations, newsletter form

## Kaise Use Karein
1. Zip extract karo
2. `index.html` browser mein open karo
3. Internet connection chahiye (images aur fonts CDN se load hote hain)

## Features
- Sticky header with nav
- Hero section with background image
- Mission banner
- Spotlight cards (with Urgent tags)
- Get Involved section (6 cards)
- Stories grid
- Donate CTA section
- Newsletter signup
- Full footer with links
- Scroll animations
- Responsive design (mobile friendly)
